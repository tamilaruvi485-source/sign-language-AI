# 🤟 SignLanguage AI

### Real-Time Sign Language to Speech Conversion System

SignSpeak AI is an AI-powered application that converts sign language gestures into text and speech in real time using Computer Vision and Artificial Intelligence. The system helps bridge communication gaps and promotes accessibility for individuals with speech and hearing impairments.

---

# ✨ Features

🔹 Real-Time Hand Gesture Detection

🔹 Gesture-to-Text Conversion

🔹 Automatic Sentence Formation

🔹 Text-to-Speech Output

🔹 MediaPipe Hand Tracking

🔹 User-Friendly Interface

🔹 Accessibility-Focused Solution

# 🚀 How It Works

1. Captures hand gestures using a webcam.
2. Detects hand landmarks using MediaPipe.
3. Identifies gestures based on finger positions.
4. Converts gestures into words and sentences.
5. Generates voice output using Text-to-Speech.
